# Jewellery CMS

A full-stack Jewellery CMS platform with a centralized admin dashboard for managing products, categories, media assets, and homepage content. Built with Next.js and MongoDB for a scalable and responsive experience.

## Live Demo

Frontend: https://davaljewels.vercel.app/
